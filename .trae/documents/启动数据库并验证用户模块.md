- 创建集成测试脚本 `scripts/verify-user-flow.ts`。
- 模拟完整流程：
  1. **注册**新用户 (验证 `AUTH-003`)
  2. **登录**获取 Token (验证 `AUTH-006`)
  3. **获取个人信息** (验证 `AUTH-010` 中间件 + `USER-001`)

- 运行脚本并确认所有 API 返回 200 OK。
